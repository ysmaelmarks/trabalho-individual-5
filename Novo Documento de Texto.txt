Vamos usar o Node para montar um código que vai receber uma lista de 
propriedades de CSS (ex: background-color, font-size, text-align) e vai devolver 
essa lista ordenada de A-Z. Ordenar o CSS ajuda a encontrar mais rápido a 
propriedade que precisamos alterar.


➔ O código deve rodar em um laço que vai receber propriedades até a 
pessoa digitar a palavra “SAIR”, quando isso acontecer vai ser impresso 
no terminal a lista das propriedades ordenadas de A-Z uma em cada linha.

1 - função
2 - laço de repetição (argumento != sair){
2a - leitura de argumento
2b - if (argumento == sair){
	stop
	}else{
2c - argumento no array (embaralhado)
	}
   }

3 - array.sort (array desembaralhado)
4 - imprimir array[i] por linha



Sem uma biblioteca de suporte que gerencie as interações com o usuário, como o readline, seria difícil implementar um laço de repetição que seja interrompido pela entrada do usuário de forma adequada na linguagem JavaScript(Node). O readline fornece uma interface fácil de usar para ler dados da entrada padrão (stdin), o que é útil para construir interações com o usuário em aplicações de console.
